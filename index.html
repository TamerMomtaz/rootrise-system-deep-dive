The Complete Workflow

Every step from user arrival to report delivery, showing exactly who does what.

1
Intake
User + Platform
1
User Arrives
User visits RootRise and selects their role (SME / Investor / Government)
2
User Configures
Selects agents (Core mandatory, Add-ons optional), report format, language, depth
3
User Completes Questionnaire
~50-80 questions across business dimensions. Uploads documents. Takes 15-30 minutes.
User's active role essentially ends here. They wait for results.
2
Intelligence Activation
Platform + &Eye
4
Data Validation (Gate 1)
System automatically checks: All fields complete? Documents valid? Data consistent?
If FAIL ‚Üí User prompted to fix. If PASS ‚Üí Continue.
5
Sector Detection (&Eye)
&Eye analyzes responses to detect: Primary industry, sub-sector, geographic context
6
Lens Activation (&Eye)
Loads sector-specific knowledge: standards, certifications, diagnostics, regulators, benchmarks
This context is INJECTED into every agent's prompt.
Agents now "see" through the industry lens.
3
Analysis
The Pantheon (10 Agents)
7
Orchestration Begins (The Drucker)
The Drucker receives config, data, lens context, and selected agents list
8
Parallel Agent Execution
All agents run SIMULTANEOUSLY. Total time: ~60-90 seconds.
  • Core: Marvin (Diagnostic), Graham (Finance)
  • Add-ons: Ricardo (Export), Lovelace (Digital), Mayo (HR), Ohno (Supply), Porter (Market)
  • Utility: Tufte (Reports), Deming (QA)
9
Agent Outputs Collected
Each agent produces: scores, findings, recommendations, confidence level, evidence
10
Confidence Check (Gate 2)
Any agent confidence < 0.7? ‚Üí Flag for human review
11
Conflict Resolution (Gate 3)
The Drucker checks for contradictions between agents. Attempts resolution, escalates if needed.
4
Synthesis
The Drucker + The Tufte
12
Narrative Synthesis
The Drucker weaves outputs into: executive summary, overall score, prioritized roadmap
13
Report Generation
The Tufte creates output in requested format with user's language/depth preferences
5
Quality & Delivery
The Deming + &I Expert
14
Quality Assurance (Gate 4)
The Deming validates: accuracy, consistency, completeness, actionability
15
HITL Decision
Human review triggered for: first-time users, high-stakes, unusual patterns, low confidence
If triggered: Expert reviews draft, validates, adjusts, approves (1-4 hours)
16
Delivery
User receives: downloadable report, dashboard access, email notification
